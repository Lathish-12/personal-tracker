Google Sign-In Setup

Activate real authentication by linking your Google Cloud OAuth Client ID.

How to obtain a Client ID:

  1. Go to the Google Cloud Console.
  2. Create a project and configure the OAuth Consent Screen.
  3. Create credentials of type OAuth client ID (Web Application).
  4. Add http://localhost:8000 to Authorized JavaScript origins.

GitHub Sign-In Setup

Link your GitHub OAuth keys to enable real user authentication profiles.

Register your app on GitHub Developer Settings:

  • Set Homepage URL to: http://localhost:8000
  • Set Authorization callback URL to: http://localhost:8000
